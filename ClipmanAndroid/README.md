# Clipman Android

Clipman Android is the Android client for Clipman Server. It lets Android users browse, search, copy, add, edit, pin, delete, and open links from a shared Clipman text history while the app is open.

For user setup and testing instructions, read `Manual.html`.

Server setup can import the private `.clpconf` connection file generated by Clipman Server, avoiding manual entry of a long address and token. Open the file from Android's file browser to send it to Clipman, or use **Import server connection file** in Clipman Settings. Server mode requires a nonblank, preferably unique history password and does not connect merely because the file was imported.

Settings can optionally require biometric or device authentication whenever Clipman returns to the foreground. This is off by default. For private-CA HTTPS servers, install the server's public authority as an Android CA certificate; Clipman opts into user-installed authorities for its server connection.

Clipman can check the official GitHub release for updates automatically while it is open, at most once per day, or on demand from Settings. It verifies the downloaded package before Android asks the user to approve installation.
