# Clipman Android

Clipman Android is the Android client for Clipman Server. It lets Android users browse, search, copy, add, edit, pin, delete, and open links from a shared Clipman text history while the app is open.

For user setup and testing instructions, read `Manual.html`.

Server setup can import the private `.clpconf` connection file generated by Clipman Server, avoiding manual entry of a long address and token. Open the file from Android's file browser to send it to Clipman, or use **Import server connection file** in Clipman Settings.
